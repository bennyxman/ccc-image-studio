**Professional Image Editing in the Browser — by Continental Capital Connect**

🔗 **Live Demo:** [https://bennyxman.github.io/ccc-image-studio](https://bennyxman.github.io/ccc-image-studio)

🌍 **Languages:** Deutsch | English | Español | Português | Français

---

## Features

| Feature | Description |
|---------|-------------|
| ◐ **Adjust** | Brightness, Contrast, Saturation, Grayscale, Sepia, Hue, Blur |
| ◈ **Filters** | 12 Presets — Vivid, Warm, Cool, Cinema, B&W, Vintage, Drama, Fade, Noir, Sharp, Amber |
| ⬔ **Crop & Transform** | Free crop, rotation (free + 90°), horizontal/vertical flip |
| ✎ **Annotate** | Freehand drawing + text placement, 8 colors, adjustable thickness |
| ▦ **Mosaic** | Pixelate regions for privacy (faces, license plates, etc.) |
| ❑ **Frames** | 8 frame styles — Thin, Classic, Shadow, Round, Polaroid, Gold, Double |
| ◎ **Watermark** | Custom text, 4 positions, adjustable opacity |
| 🪄 **Background** | Remove background by click + replace with color or custom image |

## Additional Features

- **5 Languages** — German, English, Spanish, Portuguese, French (auto-detects browser language)
- **Settings Panel** — Language selection, export format (PNG/JPEG)
- **About & Help Modals** — Built-in documentation and keyboard shortcuts
- **New Window** — Open additional editor instances
- **Undo/Redo** — Full editing history
- **Keyboard Shortcuts** — `Ctrl+Z`, `Ctrl+Y`, `Ctrl+S`, `Ctrl+V`
- **Drag & Drop + Clipboard Paste**
- **100% Client-Side** — No data uploaded, everything runs locally

## Privacy

All image editing happens **locally in your browser**. No images are uploaded or sent to any server. The app works offline after the initial font load.
