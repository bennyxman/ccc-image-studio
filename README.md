# CCC Image Studio

**Professionelle Bildbearbeitung im Browser — by BridgepointAfreuropa**

🔗 **Live Demo:** [https://bennyxman.github.io/ccc-image-studio](https://bennyxman.github.io/ccc-image-studio)

---

## Features

| Feature | Beschreibung |
|---------|-------------|
| ◐ **Korrektur** | Helligkeit, Kontrast, Sättigung, Graustufen, Sepia, Farbton, Weichzeichner |
| ◈ **Filter** | 12 Presets — Vivid, Warm, Cool, Cinema, B&W, Vintage, Drama, Fade, Noir, Sharp, Amber |
| ⬔ **Zuschneiden** | Freies Zuschneiden, Drehung (frei + 90°), Spiegeln horizontal/vertikal |
| ✎ **Annotieren** | Freihand-Zeichnung + Textplatzierung, 8 Farben, einstellbare Stärke |
| ▦ **Mosaik** | Bereich verpixeln für Datenschutz (Gesichter, Nummernschilder, etc.) |
| ❑ **Rahmen** | 8 Rahmen-Stile — Fein, Klassisch, Schatten, Rund, Polaroid, Gold, Doppelt |
| ◎ **Wasserzeichen** | Eigener Text, 4 Positionen, einstellbare Deckkraft |

## Zusätzliche Features

- **Undo/Redo** — Vollständige Bearbeitungshistorie
- **Keyboard Shortcuts** — `Strg+Z` (Undo), `Strg+Y` (Redo), `Strg+S` (Export)
- **Drag & Drop** — Bilder direkt in die App ziehen
- **Clipboard Paste** — `Strg+V` zum Einfügen aus der Zwischenablage
- **PNG Export** — Hochqualitätiver Download mit allen Bearbeitungen
- **100% Client-Side** — Keine Daten werden hochgeladen, alles läuft lokal

## Technologie

- **Standalone HTML** — Eine einzige Datei, kein Build-Prozess nötig
- **Vanilla JavaScript** — Keine Abhängigkeiten, kein Framework
- **CSS Custom Properties** — Durchgängiges Design-System
- **Canvas API** — Professioneller Bildexport
- **Google Fonts** — Playfair Display + DM Sans

## Nutzung

### Online (GitHub Pages)
Einfach den Link oben öffnen — fertig.

### Lokal
1. `index.html` herunterladen
2. Im Browser öffnen
3. Bild laden und bearbeiten

### Weitergabe
Die `index.html` Datei kann per E-Mail, USB-Stick, WhatsApp oder Cloud geteilt werden.

## Datenschutz

Alle Bildbearbeitung findet **lokal im Browser** statt. Es werden **keine Bilder hochgeladen** oder an Server gesendet. Die App funktioniert auch offline (nach dem ersten Laden der Schriftarten).
